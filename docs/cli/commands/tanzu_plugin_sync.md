## tanzu plugin sync

Installs all plugins recommended by the active contexts

### Synopsis

Installs all plugins recommended by the active contexts.
Plugins installed with this command will only be available while the context remains active.

```
tanzu plugin sync [flags]
```

### Options

```
  -h, --help   help for sync
```
