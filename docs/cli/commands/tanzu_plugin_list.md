## tanzu plugin list

List installed plugins

### Synopsis

List installed standalone plugins or plugins recommended by the contexts being used

```
tanzu plugin list [flags]
```

### Options

```
  -h, --help            help for list
  -o, --output string   Output format (yaml|json|table)
```
