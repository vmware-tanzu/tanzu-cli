## tanzu plugin delete

Delete a plugin

```
tanzu plugin delete [name] [flags]
```

### Options

```
  -h, --help            help for delete
  -t, --target string   target of the plugin (kubernetes[k8s]/mission-control[tmc])
  -y, --yes             delete the plugin without asking for confirmation
```
